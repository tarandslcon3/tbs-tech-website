TBS Tech Services — Content Rules (Read Before Writing Anything)

- SEO and AI visibility (ranking on Google and getting recommended by ChatGPT, Perplexity, Claude)

Anti-Hallucination Rules (Non-Negotiable)

- Never invent statistics. If you cite a number, it must come from a real search result in this session.
- Never name a specific tool price unless you searched and confirmed it today.
- Never claim TBS Tech Services has X clients or X years experience — we do not state these publicly.
- Never invent client names, case studies, or testimonials unless they are the placeholder ones already on the site.
- Never say "studies show" without citing the actual source found in search.
- If unsure about a fact — write around it or say "typically" or "most businesses find."
- Before writing any blog post, search for the topic first. Write from real search results, not memory.

Formatting Rules

- Use simple plain bullet points for lists and steps. No emoji bullets. No fancy symbols. Just a hyphen dash.
- Use numbered lists for steps that must be done in order.
- Every step must be complete. Never write "set up your account" without explaining exactly how.
- Bold only for headings and genuinely important warnings. Never bold random words mid-sentence.
- No AI-sounding phrases like "In today's fast-paced world" or "Navigating the digital landscape" or "Leverage synergies."
- No filler sentences. Every sentence must add information or value.
- Write like a knowledgeable friend explaining something over coffee. Casual but accurate.

Blog Post Rules

- Minimum 1200 words for every post.
- Target one specific problem real people search for. Check Google Trends or search results before picking a topic.
- Structure every post exactly like this:

1. Hook — one paragraph hitting the exact pain point. Make the reader feel understood.
2. Why this problem exists — context so the reader knows they are not alone or dumb for not knowing this.
3. The DIY solution — complete step by step. Numbered steps. Every step explained fully. No skipping.
4. What tools you need — list them with honest free vs paid options.
5. Common mistakes to avoid — real ones, not generic warnings.
6. When to hire a professional instead of DIY — position TBS Tech Services naturally here, not as an ad.
7. FAQ section — 5 questions written conversationally for AEO. These get picked up by ChatGPT and Google AI.
8. Closing with soft CTA to TBS Tech Services free demo.

- Every post needs: meta title under 60 chars, meta description under 160 chars, H1 H2 H3 structure, read time estimate, author TBS Tech Services Team, date, featured image placeholder.
- Internal links: every post must link to /calculator and to the contact section.
- After writing, add to blog listing page and update app/sitemap.js.

Social Media Rules

LinkedIn Posts

- Hook line first — one sentence that makes someone stop scrolling.
- Then the problem explained in plain words.
- Then the tip or solution — real value, not vague advice.
- Soft CTA last — never pushy.
- Under 200 words total.
- No hashtag stuffing — maximum 3 relevant hashtags at the end.
- Reads like a human wrote it. If it sounds like AI wrote it, rewrite it.

Google Business Posts

- Under 300 words.
- One specific tip or piece of advice.
- Written for local business owners and small businesses.
- Always end with: tbstechservices.vercel.app

Facebook Posts

- Under 150 words.
- Casual and conversational.
- Always ends with a question to drive comments.
- Never promotional in tone.

Content Review Agent Rules

Before finalizing any blog post or article, the agent must check all of the following:

- Does this post actually help someone solve a real problem? If no, rewrite.
- Does every step actually work? Would a non-technical person be able to follow it? If no, add more detail.
- Is there any invented statistic or unverified claim? If yes, remove it or find the source.
- Does the post mention a TBS Tech Services service that is relevant? If not, add one natural mention.
- Is the post skimmable with clear headings and bullet points for steps? If no, restructure.
- Does the post end with a genuine CTA that matches what the post is about? If no, fix it.
- Does the post have the FAQ section with 5 AEO-optimized questions? If no, add it.
- Is the meta title under 60 characters targeting a real keyword? If no, fix it.
- Is there any sentence that adds zero value? If yes, delete it.
- Does the post link to the calculator and contact section? If no, add links.

Daily Content Workflow

Every day before writing content:

1. Search for what small business owners are struggling with today — use real search results.
2. Pick the topic with highest search relevance to TBS Tech Services.
3. Write the blog post following the structure above.
4. Run the content review agent checklist above on the finished post.
5. Fix anything the checklist flags.
6. Then write the three social posts based on the same topic so everything is consistent.
7. Push blog post to main branch.
